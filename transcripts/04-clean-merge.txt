
================================================================
  SCENARIO 4 — concurrent inserts on DIFFERENT rows → clean three-way merge
================================================================

---- Clone A adds row id=10; commits locally; does NOT push yet ----
(A has its own commit; remote unchanged)

---- Clone B adds row id=20; commits; pushes first — wins the race ----

$ dolt add items
